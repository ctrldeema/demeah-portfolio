Deemah AlBabtain Portfolio Website

How to use:
1. Open index.html in any browser to preview the website locally.
2. To publish, upload the whole deemah_portfolio folder to GitHub Pages, Netlify, or Vercel.
3. Keep the assets folder next to index.html so images and the CV link work correctly.

Main files:
- index.html: website structure and content
- styles.css: visual design and responsive layout
- script.js: navigation, image preview modal, and scroll reveal interactions
- assets/: project screenshots, CV PDF, and supporting visuals
